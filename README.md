# HireMe-Clone

## Description
HireMe.com is a database of resume, jobs, and recruitment consultants. Conceived as a platform of job seekers and hiring managers .

## Tech Stack

- **Javascript**
- **React**
- **Redux Thunk**
- **Axios**
- **Chakra-UI**
- **JSON Server**
